# Social Lab V1 PRD Summary

## Product Definition

Social Lab is an AI interpersonal communication simulation platform. It helps users rehearse difficult conversations before they happen by generating a target-person persona, simulating likely responses, predicting outcomes, and coaching the user's expression.

## Slogan

先演练，再开口。

Practice difficult conversations before they happen.

## V1 Product Goal

V1 focuses on validating whether users find value in rehearsing communication with a specific simulated person before contacting the real person.

## Target Users

- University students handling advisor communication, recommendation letters, feedback requests, and deadline extensions
- Early-career professionals handling upward communication, workload negotiation, and compensation conversations
- Socially sensitive users handling rejection, apology, conflict, and relationship-boundary conversations

## Core MVP Flow

1. User selects a communication scenario.
2. User describes the communication goal.
3. User creates a target person through manual description or pasted chat context.
4. System generates a Persona Card.
5. System generates Relationship State.
6. User enters simulated conversation.
7. AI target person responds according to Persona, State, Memory, and scenario.
8. User ends the simulation.
9. System generates outcome prediction and communication coaching.
10. User copies an optimized rewrite or retries the simulation.

## V1 Scope

Included:

- Landing page
- Scenario selection
- Target-person creation
- Persona Card
- Relationship State
- Simulated chat
- Outcome prediction
- Communication coaching
- Privacy reminder
- Responsive Web MVP

Excluded:

- Native app
- Long-term memory
- Multi-person relationship graph
- Social features
- Real contact syncing
- Automatic message sending
- Medical, legal, or therapy advice

## Current Repository Status

The current implementation is a static front-end prototype. It validates the user journey and UI structure before adding real AI, backend APIs, and persistence.
